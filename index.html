About INT

INT is a free, open source CSS framework that provides ready-to-use utility classes and frontend components that you can easily combine to build responsive web interfaces.

A utility-first CSS framework packed with classes like flex, p-4, text-center and w-80 that can be composed to build any design, directly in your markup.
